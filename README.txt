IOACO

The programa has been coded in C language under Linux. 

In order to compile iOACO, you just need to type the following:

$ gcc -o ioaco -O2 IOACO.c c_files/*.c -lm

Hence, the executable program will be called: ioaco.

In order to execute it, you just need to type:

$./ioaco path/to/config/file [numberOfExecutions]

Examples of the configuration files are located at the directory config/
